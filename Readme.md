Vtalk IM

# 强大的在线聊天系统功能齐全,单聊,群聊,视频聊天,多人语音房聊天,直播, 不依赖任何第三方,只需要服务器即可搭建十万级用户在线im聊天系统,免除第三方交费封禁烦恼。不限制客户端登录数量，任意账号可多次登录同一客户端

# 可集群水平扩展部署,百万级在线不是梦


mysql,redis,mongo,rabbitmq,rtc,protobuf,rtmp,socket

用户系统

socket连接系统  
流媒体系统

后台系统
